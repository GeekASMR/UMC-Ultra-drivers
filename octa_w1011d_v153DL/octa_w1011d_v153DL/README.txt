Installation
 1. Disconnect all USB devices except keyboard and mouse, and then start Windows.
 2. Double-click the Setup icon.
 3. If a dialog box regarding User Account Control appears, click [Yes].
 4. If the message "The operation completed successfully." appears, click [OK].
 5. Connect the unit to your computer with a USB cable, and turn on the unit.
    The driver will be installed automatically.

To Uninstall
 1. Disconnect all USB devices except keyboard and mouse, and then start Windows.
 2. Exit all applications before you begin the uninstallation.
   * Please wait for about 10 seconds until these applications exit completely.
 3. Double-click Uninstall.
 4. If a dialog box regarding User Account Control appears, click [Yes].
 5. Follow the instructions in the dialogue boxes to proceed.
 6. When the message "Uninstallation was completed." is displayed, click [OK] to restart Windows.

Opening online document
 When the driver installation is completed, you can open the Readme file by following the procedure described below.
 Refer this document for troubleshooting and for checking how to set up the driver.
 1. Click the start button and then click "All apps".
 2. Open the Roland folder and then click "Readme". 
   * For BOSS products, open the BOSS folder.
 3. The online document for the driver is displayed.


■インストールする
 1. キーボード、マウスを除く、すべての USB 機器をはずした状態で Windows を起動します。
 2. Setup をダブルクリックします。
 3. ユーザー アカウント制御に関するダイアログが表示された場合は、[はい] をクリックします。
 4. 「この操作を正しく終了しました。」と表示された場合は、[OK] をクリックします。
 5. 製品とコンピューターを USB ケーブルで接続して製品の電源を入れます。
 　 自動的にドライバのインストールが行われます。

■アンインストールする
 1. キーボード、マウスを除く、すべてのUSB機器をはずした状態で Windows を起動します。 
 2. アンインストールを行う前にすべてのアプリケーションを終了してください。 
   ※ アプリが完全に終了するまで、10秒程度お待ちください。 
 3. Uninstall をダブルクリックします。
 4. ユーザー アカウント制御に関するダイアログが表示された場合は、［はい］をクリックします。 
 5. 画面の指示に従って、アンインストールを進めます。
 6. 「アンインストールを完了しました。」と表示されますので、［OK］をクリックして Windows を再起動します。

■オンラインドキュメントを参照する
ドライバのインストールが完了すると、以下の手順で README ファイルを参照することができます。
ドライバの設定方法や、使用上のトラブルに関する解決策など参照することができます。

 1. スタート・ボタンをクリックします。
    ※「すべてのアプリ」が表示された場合は、「すべてのアプリ」をクリックします。
 2. Roland フォルダを開き、「Readme」をクリックします。
    ※ BOSS 製品の場合は、BOSS フォルダを開いてください。
 3. ドライバのオンラインドキュメントが表示されます。
