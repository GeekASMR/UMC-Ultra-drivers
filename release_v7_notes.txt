# ASIO Ultra v7.1.4

ASIO Ultra 是一款专业级虚拟音频路由与 ASIO 驱动桥接工具，专为解决多通道音频分配和低延迟处理而设计。

## 最核心更新日志 (v7.1.4)

### 1. 授权激活验证修复 (关键修复)
- 修复了部分 Windows 7 及老旧 Windows 10 系统由于网络协议兼容性导致的“激活失败”问题（代码层强制启用 TLS 1.2 兼容）。
- 优化了控制面板状态同步机制，若管理员在后台进行解绑操作，本地界面会立刻响应并更新状态，防止显示异常。

### 2. 系统卸载与清理优化
- 改进了内部的清理机制，卸载时会全面扫描并移除旧版本驱动遗留的注册表残留。
- 采用智能白名单系统识别真实音频硬件，自动隐藏无效音频端点，避免干扰客户操作。

### 3. 底层接口与名称规范
- 修复了此前在音频宿主 (DAW) 中设备名称显示异常的问题，统一采用标准化的设备命名方式。
- 重构了基于原生 API 的通信管道，降低被拦截和系统杀软误杀的风险。

### 4. 音频时钟同步与防止爆音
- 改进了 WDM 与 ASIO 底层的环形共享内存同步模型，优化了数据缓存自适应填充机制。
- 大幅减少在超低延迟运行下的微弱咔音与电音，提供更稳定顺滑的录音/直播音频通路。
