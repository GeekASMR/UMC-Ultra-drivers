# 终极虚通道引擎跃迁 (V7.0.8)

🌟 **V7.0.8 完美零爆音（Zero-Click）IPC核心重构**
1. **彻底解决在 128 Buffer Size 极低延迟下持续出现的微弱咔哒声 / 细微爆音**：重写了 WDM 与 ASIO 底层环形共享内存 (Ring Buffer) 的同步模型。彻底移除了原先会导致微小重采样的“动态拉伸补偿 (Sample Stuff/Drop)”机制，全面换用极为强悍的“Prefill-Cushion（空载拉垫）”自适应缓存池系统。现在引擎会在底层开启瞬间安全地预蓄 50ms 纯净缓冲，完美吸收 Windows 毫无规律的 WDM 偶发断流，并在运行中做到极致顺滑 1:1 位对位拷贝，声音如丝般润滑清澈！

🌟 **V7.0.7 架构级立体声对齐热修复**
1. **彻底修复原厂奇数通道（Odd-Channels）导致的立体声声相漂移与错位问题**：针对如 Avid MBOX Studio 等提供奇数通道（例如包含单个 Talkback 麦克风轨道导致输出总数为 21）的专业音频接口，重写了 ASIO 内核通道对齐逻辑。现在系统会自动计算并填补 Dummy 对齐占位符（`--- Unused Alignment ---`），强制确保虚拟通道（Virtual IN/OUT）永远从绝对偶数对齐（例如序号 22）开始挂载，根除由于宿主 DAW 默认两两编组导致的 Left/Right 轨串音与错位现象！

🌟 **V7.0.6 最新兼容升级与修复**
1. **新增 Avid MBOX 原厂驱动兼容**：重构了底层的代理驱动实现架构，彻底隔离独立的接管 CLSID 链路，实现完美接入，防止覆盖或者干扰原官方驱动独立运行。

🌟 **V7.0.5 紧急维护分支更新**
1. **诊断日志无法上传修复**：修复由于 V7 核心将品牌重命名为 `ASIO Ultra` 后与早期日志收集器的临时映射错位问题，此错位曾导致后台诊断日志无法更新为用户的最新实时调试捕获档案。现在所有诊断追踪流已恢复完全同步。

*V7.0.4 之前版本所有极客务必拉取！包含之前的所有底层脱壳大修！*

🌟 **V7.0.4 终极 COM 底层架构脱壳升级 (同步集成)**
- 彻底粉碎了部分用户由于宿主运行降权而导致的 `无法初始化` (100% 根除)
- 修复了硬编码 CLSID 的 `CLASS_E_CLASSNOTAVAILABLE` 严重互斥漏洞，实现15大品牌完全自由接管和瞬间同步挂载！
