# ASIO Ultra v7.1.4

ASIO Ultra 是一款专业级虚拟音频路由与 ASIO 驱动桥接工具。本次更新注重于提升驱动的稳定性、兼容性以及安装体验。

## 更新亮点

### 1. 提升网络连通性与激活兼容性
- 修复了在部分较旧的 Windows 10 及 Windows 7 系统上因不支持最新安全协议 (TLS 1.2) 而导致的无法激活问题。
- 优化了底层网络通信通道，避免被安全软件误报，确保在各种网络环境下的激活稳定性。
- 增强了客户控制面板的同步校验逻辑，后台解绑或重置设备后，客户端界面会即时变更授权状态。

### 2. 优化驱动卸载机制
- 改进了驱动卸载逻辑，卸载时会自动扫描并清理与本驱动相关的遗留注册表项，确保卸载后系统环境纯净，不留冗余项。

### 3. 提升设备识别准确度
- 优化了设备识别的白名单机制，自动过滤虚拟声卡或不相关的音频设备，使路由分配更精准稳定。
- 音频宿主软件 (DAW) 中显示的设备名称更简洁、标准化。

### 4. 优化后台面板与音频内核
- 控制面板支持独立自适应调用，无需依赖特定宿主路径，交互响应更迅速。
- 增强了音频重采样引擎，进一步降低了不同采样率下的混音抖动和杂音问题。

## 支持的硬件设备
本版本为以下主流音频接口提供低延迟路由与虚拟跳线支持：
- Behringer UMC 全系 (U-Phoria)
- Audient / EVO 全系
- Solid State Logic (SSL 2 / SSL 2+ / SSL 12)
- Mackie (Onyx)
- TASCAM
- YAMAHA / Steinberg
- MOTU (M2 / M4)
- PreSonus (Studio USB)
- Focusrite (Scarlett 家族)
- ZOOM 系列
- ART 系列
