# V7.1.0 史诗级重构与净化

### 1. 极致底层净化 (Uninstaller 毁灭级优化)
重新设计了 DllUnregisterServer 的卸载机制。它不再死板识别固定字眼，而是开启「全图扫描雷达」，一键扫荡拔除带有当前底层驱动 UUID（CLSID）的任何变种衍生通道（如 (Trial) 等幽灵注册表节点），告别无限套娃，让强迫症极致舒爽。

### 2. 原厂白名单阵列 (Whitelist Architecture)
废弃臃肿的硬编码拉黑池。重构主脑安装包，采用极其高贵的**「国际大厂白名单验证规则」**。只要不是在内置核心靶名单内的野鸡或虚拟品牌（Realtek, Voicemeeter, Toping 等），全部自动抹杀隐藏资格，大幅简化后续安装与维护。

### 3. 纯净原味防篡改命名
宿主设备注入分配更地道。修复了自动在通道名字后画蛇添足加厂商品牌的判定规则，现在宿主名字将被绝对无痕映射，只留下至简的「Ultra」标记。

### 4. 杀软级通讯大穿透 (Native WinHTTP API)
摒弃挂载 curl.exe 的暴力通讯手段（此法易遭原生安全中心及 DAW 高级沙盒反杀）。现已 100% 重构基于原生 WinHttp 底层的 C++ 通信管道，弹窗秒开、无感过检，彻底消除弱网下离奇的激活失败误杀！


# 🌌 ASIO Ultra v7.0.2 全品牌制霸时代降临

历经长期淬炼与推翻重构，**ASIO Ultra V7.0.2** 现已正式作为全平台独立架构启航！

## 💥 V7.0.2 重大性能与兼容性修复
- **独立控制面板自适应定位系统**：使用纯 C++ 动态获取 `DllMain` 所处之内存绝对空间路径，取代过往寻找指定宿主环境的行为。无论宿主是否改名、程序置于系统何处，皆可秒弹专属低延迟暗屏调音面板。
- **老旧声卡与多指纹防封解卡**：修正在读取注册表 `HKEY_LOCAL_MACHINE` 映射时对如遗留版 UM2 `BEHRINGER USB AUDIO` 接口等短名识别锁死的崩溃。完美识别并截获任意宿主对多变体的声卡拦截控制。
- **/MT 强劲提纯与体积缩减**：完全脱离 VC++ 系统依赖环境单独装载运行。废除多余后台模块。纯净精简至极致！

## 🚀 核心爆音隐患终极抹除 (Zero-Tearing SRC)
- **极寒零杂音 SRC 引擎**：彻底重构底层 `writeStereoSRC` 与 `readStereoAdaptive` 同步拦截机制。无论您的宿主 (DAW) 是否以 44.1kHz 运作，内核均可通过超精准的亚帧小数级时间锁定追踪补偿漂移，不再有跨缓冲区的任何撕裂断流现象，完美消除所有的时钟咔音与电流噪音。
- **原生 48k 流空拦截满推**：全自动补足 WDM 内核音频驱动的底层跳频机制限制，动态缓冲锁定，实现真正的纯净无忧虚实交火。

## 🎯 系统通道深度隔离内嵌化
全面告别额外的执行环境补丁与外链小程序依赖！现在，**系统通道深度隔离架构**已百分之百原生深度内嵌至控制台 DLL 极速层中。只需在驱动管理面板点击原生按钮，高权限调度即可立刻接管所有通道底层。
